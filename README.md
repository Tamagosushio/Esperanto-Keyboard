# Esperanto-Keyboard
## Overview
エスペラント語をキーボードで打てるようにします
## Usage
1. `EsperantoHook.dll`と`EsperantoLoader.exe`をダウンロードする
2. 同じディレクトリに置く
3. exeファイルを起動して有効化
- 主な機能として、字上符付き文字をx,Xを用いて打てます
  - cx -> ĉ
  - gX -> ĝ
  - Hx -> Ĥ
  - JX -> Ĵ
## Requirement
- Windows
